Select data from the population and country_mapping tables to find the top ten subregions with the most significant change in population from 2010 to 2020. Create a horizontal bar chart to visualize the results.

Steps breakdown:

  • Output table needs to have these columns: Region, SubRegion and Sum of PopChange
  • We need to group data by Region and SubRegion
  • Filter the data for only 2010 to 2020
  • Sort the Sum of PopChange column and retreive only the top 10 sub-regions with most significant change in population during those years
  • Create a horizontal bar chart with SubRegion on Y-axis and sum of PopChange on X-axis
In[8]:
# Define SQL query
query = '''
        SELECT cm.Region, cm.SubRegion, SUM(p.PopChange) AS PopChange
          FROM country_mapping AS cm
          JOIN population AS p
            ON cm.CountryCode = p.CountryCode
         WHERE p.Year BETWEEN 2010 AND 2020
         GROUP BY cm.Region, cm.SubRegion
         ORDER BY PopChange DESC
         LIMIT 10;'''
# Execute the query and read results into a dataframe
results = pd.read_sql_query(query, conn)
print(results)

# Create horizontal bar plot
plt.barh(results['SubRegion'], results['PopChange'])
plt.xlabel('Population Change')
plt.ylabel('Subregion')
plt.title('Top 10 Subregions by Population Change from 2010 to 2020')
plt.show()
