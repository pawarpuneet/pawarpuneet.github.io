What to study to win jeopardy?¶

In order to figure out whether to study past questions, study general knowledge, or not study it all, it would be helpful to figure out two things:

  1. How often the question can be used for a answer?
  2. How often questions are repeated?

We can answer the first question by seeing how many times words in the answer also occur in the question. We can answer the second question by seeing how often complex words (> 6 characters) reoccur.

1. How often the question can be used for a answer?¶

Do I need to study at all or can i just find the answer within the question itself? Let's find out. We want to ignore common words like 'the' which doesn't have much significance for our goal.

In [10]:
# define a function that checks for each word (except for any words like 'the') in the answer and 
# tries to see if the same word is also present in the corresponding question.
# if there is a match then match count is increased by 1
# once we have the final match count we divide it by the total number of words we have in the answer to get the proportion of match.

unwanted_words = {'the'}
def calc_match_proportion(row):
    split_answer = row['answer'].split(' ')
    split_answer = [x for x in split_answer if x not in unwanted_words]
    if len(split_answer) ==0:
        return 0
    split_question = row['question'].split(' ')
    match_count = 0
    for w in split_answer:
        if w in split_question:
            match_count += 1
    return match_count/len(split_answer)

# For every word in a question whihc is greater than 6 characters long, check if that word has occurred in the past questions. 
# If it has then increase the match count by 1
# Divide the total match count for each question by length of word list in a question. This is question_overlap proportion
# Maintain a set of terms used in all the past questions and 
# also a counter for calculating frequency counts. this will be useful later.
terms_used = set()
terms_count = Counter()
question_overlap = []
for index, row in df_jeopardy.iterrows():
    split_question = row['question'].split(' ')
    split_question = [w for w in split_question if len(w) >=6]
    match_count = 0
    for w in split_question:
        if w in terms_used:
            match_count += 1
    terms_used.update(split_question)
    terms_count.update(split_question)
    if len(split_question) !=0:
        question_overlap.append(match_count/len(split_question))
    else:
        question_overlap.append(0)

df_jeopardy['quesion_overlap'] = question_overlap

Which topics should I study to win more money?¶

I want to study questions that pertain to high value questions instead of low value questions. This will help me earn more money when I am on Jeopardy. I can figure out which terms correspond to high-value questions using a chi-squared test. I will first need to narrow down the questions into two categories:

Low value -- Any row where Value is less than or equal to 800.
High value -- Any row where Value is greater than 800.

Conclusion¶

I have explored the data to come up with a strategy to win jeapordy.

  1. It is rare that a question itself contains the answer. On an average only 6% of the words in answer also occured in the question. So I have to study for sure.
  2. On average about 70% of the words in newer questions are also present in past questions. We looked at individual words not at phrases. So even though, it doesn't tell us if same question was repeated, it tells us there is repetition and it will be worth studying the questions from the past.
  3. There are certain words which are associated with higher value questions. From the limited sample, we found two such words -'italian' and 'german'. This implies that possibly country specific questions tend to be associated with higher winning amount. And there could more such topics which are can narrowed down with further analysis.
