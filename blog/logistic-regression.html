Logistic Regression¶

Let's explore the following for classification:

  1. sklearn's Logistic Regression using 'lbfgs' solver
  2. Logistic Regression using Stocastic Gradient Descent
  3. Classification result metrics - accuracy score, confusion matrix and classification report

We will follow it up with an example problem where we will perform:

  • linearity checks on the data
  • fit a classification model and
  • analyze the scoring metrics

data_plotting.plot_multicollinearity_checks(X)

          The Pearson correlation coefficient measures the linear relationship between two datasets.
          Test for the statistical significance of the correlation coefficient:
          ρ = population correlation coefficient
          Null hypothesis (H0): ρ = 0 populations are linearly uncorrelated 
          Alternative hypothesis (H1): ρ ≠ 0 populations are linearly correlated 
          p-value < 0.05 => reject H0

    Each independent variable is tested against the rest to calclulate VIF(Variance Inflation Factor).
    VIF for each independent variable:      
-------------------------------------------------
   VIF      feature          result
0  1.0  exam1_score  Not correlated
1  1.0  exam2_score  Not correlated

    Here is how it can be interpreted :
    1= Not correlated
    1–5 = Moderately correlated
    >5 = Highly correlated

Logistic Regression¶

In [27]:
X_train, X_test, y_train, y_test = train_test_split(X,y,test_size=0.2)
pipe = Pipeline([('pr',PolynomialFeatures(include_bias=False,degree=2)),('scaler',StandardScaler()),('clf',LogisticRegression(penalty='l2',C=1.0))])
pipe.fit(X_train, y_train)
cv = RepeatedKFold(n_splits=5,n_repeats=5,random_state=1)
scores = cross_val_score(pipe,X_train,y_train,cv=cv,scoring='accuracy')

scores.mean(), scores.std(),pipe.score(X_test,y_test) # accuracy
