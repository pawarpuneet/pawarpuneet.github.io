Q-Q plot

Q-Q plot is a very useful tool when checking for normality. Let's understand stepwise how it is created, what different types of reference lines are used and how to interpret them. Aftet that, we will also implement the quickest method for it.

plt.scatter(x=df['theoretical_quantile'],y=df['data'])

#add Q-Q line, it is NOT y=x line
q1_obs = stats.scoreatpercentile(a=df['data'], per=25)
q3_obs = stats.scoreatpercentile(a=df['data'], per=75)
q1_theo = stats.norm.ppf(.25)
q3_theo = stats.norm.ppf(.75)
slope = (q3_obs - q1_obs) / (q3_theo - q1_theo)
intercept = q1_obs - slope * q1_theo # or q3_obs - slope * q3_theo

print(slope, intercept)

plt.axline((0,intercept), slope=slope)

plt.show()

Best quick method¶

Use qqplot from statsmodels.api

Few options for Q-Q line:

  • You can use line = '45' and it will work well if you have z-normalized data, meaning your distribution will have mean = 0 and sd = 1.
  • Use line = 's' in case you want to see a fit against standardized line (the expected order statistics are scaled by the standard deviation of the given sample and have the mean added to them)
  • line = 'q' line fit through the quartiles (q1 and q3), which is the one really meaning full and lets us observe well the deviation of the data distribution from the normal one.
  • Also, you can use line = 'r' for to see the fit to regression line.
In [6]:
import statsmodels.api as sm
import matplotlib.pyplot as plt
sm.qqplot(df['data_norm'], line='45') # '45' is y=x line (uncomment the code below to confirm)

In [7]:
sm.qqplot(df['data'], line='s')

plt.show()

The reference line in a QQ plot is typically drawn by connecting the first and third quartiles of the observed data and the theoretical distribution, rather than being a y=x line, because this method provides a robust and quick visual assessment of the data's distributional fit. This approach ensures the line is based on the central part of the distribution, making it less sensitive to outliers or extreme values in the tails. The line's slope and intercept are calculated using the quantiles of the data and the theoretical distribution, resulting in a line that reflects the specific characteristics of the sample, such as its mean and standard deviation, rather than assuming a perfect match to a standard normal distribution.
